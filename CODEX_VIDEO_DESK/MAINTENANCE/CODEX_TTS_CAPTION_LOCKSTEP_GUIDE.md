# Codex TTS-Caption Lockstep

## 목적

영상의 나레이션과 화면 청크가 서로 다른 내용을 보여 주는 문제를 차단합니다.

## 적용 규칙

- 화면 청크는 해당 구간 TTS 원문을 순서대로 빠짐없이 분할합니다.
- 화면 문구용 별도 요약문은 만들지 않습니다.
- 화면에서 마침표는 숨기되 단어를 추가하거나 삭제하거나 바꾸지 않습니다.
- 분할 우선순위: 문장 끝 → 쉼표 → 연결 어미 → 한국어 호흡.
- 공백 제외 9~27자를 우선하고 최대 34자를 넘기지 않습니다.
- 기존 고정 CTA, 원본 이미지 1회 사용, 일러스트 균형, TTS 발음 사전은 유지합니다.

## 설치

`RUN_APPLY_CODEX_TTS_CAPTION_LOCKSTEP.bat`

설치 후 기존과 동일하게 `CODEX_VIDEO_DESK`의 렌더 버튼을 사용합니다.
