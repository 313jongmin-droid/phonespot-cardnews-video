# 폰스팟 사용법 — 빠른 시작

독립 로컬 생산기 기준. 각 PC가 혼자 다 합니다. 공유는 일러스트 라이브러리만 옵션.

## 매일 한 편 만들기 (한 PC에서)

1. **패널 켜기**: `00_PHONE_SPOT_PANEL.bat` (헤더 우측 버전 배지로 최신 확인).
2. **카드뉴스 만들기**: `cardnews\webui\start.bat`(웹UI) 또는 `cardnews\run_all.bat`.
3. **영상 모드 → "1. 영상용 프롬프트 준비"**: 스크립트·일러스트 요청 생성.
4. **이미지 채우기**(필요할 때만):
   - 영상 일러스트: GPT로 그려 다운로드 → **"2. 이미지 가져오기 + 렌더"** → 검수창에서 자동배정 확인 → 확정.
   - 카드 이미지: 카드뉴스 모드 → **"3-2. 이미지 자동 배정(검수)"** → 확인 → `1~5.png`.
5. **렌더**: 이미지 있으면 **"2"**, 없으면 **"3. 선택 영상만 렌더"**.
6. **결과**: **"영상 결과 폴더"** 에서 mp4 회수.

> 라이브러리가 충분히 쌓이면 4번은 점점 필요 없어집니다(검수창이 "후보 없음/바로 렌더").

## 일러스트 라이브러리 공유 (협업할 때만)

목적: PC끼리 같은 그림 또 안 그리게, 라이브러리를 함께 쌓아 수렴시키기.

1. **최초 1회 — 허브 경로 지정**(공유폴더/드라이브/USB 중 한 곳):
   - 명령창: `setx PHONESPOT_LIBRARY_SHARE "G:\공유\phonespot_library"`  (각 PC에서)
   - 또는 파일 `shorts\config\library_share_path.txt` 에 경로 한 줄.
2. **동기화 실행**: 패널 **"일러스트 라이브러리 공유"** 버튼 → 실행 로그에 가져옴/올림 개수 표시.
   (또는 `라이브러리_공유_동기화.bat`. 미리보기는 `--dry-run`.)
3. **언제**: 새 그림 확정 뒤, 또는 하루 한 번. 양방향이라 **순서 무관**.

주의: 같은 파일명은 안 덮습니다(비파괴). 드라이브 허브면 동기화 끝난 뒤 실행.

## 새 PC 합류 (1회 설치)

1. `00_SETUP_RENDER_PC_FROM_GITHUB.bat` (코드 + 렌더 의존성).
2. `shorts\SETUP_EMBED.bat` (임베딩 모델).
3. `00_PHONE_SPOT_PANEL.bat` 실행. (공유할 거면 위 허브 경로 지정.)
   - 카드 생성까지 하려면 `cardnews\webui\start.bat` 도 확인.

## 자주 쓰는 스위치 / 규칙

- **패널 코드 바꾼 뒤**: `server.py`의 `PANEL_VERSION`과 `start_hidden.ps1`의 `$expectedVersion`을
  같이 한 칸 올리고 `00` 재실행 → Ctrl+Shift+R. 헤더 배지로 반영 확인.
- **렌더 속도 테스트**: `setx PHONESPOT_RENDER_CONCURRENCY 50%` / GPU `setx PHONESPOT_GL angle`
  (워커/패널 재시작). 자세히는 `MAINTENANCE\PHONESPOT_UPDATES_2026-06_GUIDE.md`.
- **로그 줄 때**: 실행 로그의 **"로그 복사"** 버튼.

자세한 동작·한계: `MAINTENANCE\MULTI_PC_STANDALONE_AND_LIBRARY_SHARING_GUIDE.md`,
