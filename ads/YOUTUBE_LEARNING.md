## cardnews/shorts 작성 룰 (필수 적용)

### 스크립트 작성 전 (반드시)
1. 시트 "유튜브_인사이트" 탭 Read (또는 가장 최근 캐시 활용)
2. 다음 정보 추출:
   - **Top 키워드 top 5** — 제목·도입부 자연스럽게 반영
   - **후킹 패턴 top 3** — 의문/숫자/감탄/시간/가격 등
   - **우수 영상 공통점** — 다음 스크립트의 톤·구조에 반영
   - **저조 패턴** — 회피
   - **권장 룰** — 직접 적용

### 적용 우선순위
- **제목 (가장 중요)**: Top 키워드 1~2개 + 후킹 패턴 1개
- **도입부 (첫 1.5초)**: 우수 영상 공통점 따라 작성
- **본문**: 키워드 자연스럽게 산재

### 절대 하지 말 것
- 저조 패턴 답습 (인사이트 시트 "저조 영상 패턴" 섹션 참조)
- Top 키워드 강제 욱여넣기 (의미 어색하면 X)

## Gemini API 설정 (선택)

기본은 폴백 (단순 공백 분리) 작동. 더 정확한 분석을 위해 Gemini API 활성화:

1. Google AI Studio (https://aistudio.google.com/) → API Key 발급
2. Apps Script 에디터 → 프로젝트 설정 → 스크립트 속성
3. 키 추가:
   - 이름: `GEMINI_API_KEY`
   - 값: `<발급받은 키>`
4. 다음 generateYouTubeInsights 실행 시 자동 Gemini 분석

### Gemini 사용 시 vs 폴백
| 항목 | Gemini | 폴백 |
|------|--------|------|
| Top 키워드 | 한국어 명사 정확 추출 (조사/어미 제거) | 공백 분리, 조사 일부 남음 |
| 후킹 패턴 | 의미 기반 (시간/가격/공감 등) | 정규식 기반 (숫자/의문문만) |
| 공통점 분석 | 의미 기반 추출 | "(비활성)" 표시 |
| 권장 룰 | 직접적 조언 | 일반 메시지 |

→ Gemini 무료 tier 일 1500 request 충분 (하루 1회 사용).
