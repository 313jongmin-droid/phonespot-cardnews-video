## KT 가이드라인 우선 원칙

KT 공식 인증 대리점이므로 다음 영역에서 KT 본사 가이드라인 위반 시 **계약 위험**:

| 영역 | KT 가이드라인 적용 |
|------|------------------|
| 광고 카피 | 허용 문구·금지 문구 명시 |
| 로고 사용 | KT 본사 로고 가이드 (위치·크기·색상) |
| 키워드 | 허용 키워드만 (`KT 직영`, `공식 대리점` 등). `성지`, `최저가` 등 가격 부각 금지될 수 있음 |
| 색상·디자인 | KT 메인 컬러 + 표준 폰트 |
| 사전승낙서 | KT 본사 발급 사전승낙서 첨부 의무 |
| 가격 표시 | 정확성 강제. 부풀리기·과장 금지 |

→ **광고 집행 전 KT 본사 측 검수 필요할 수 있음.** 검수 이력은 시트의 별도 탭에 기록.

| 채널 | 광고 관리자 | 입력 위치 |
|------|------------|----------|
| 메타 | https://adsmanager.facebook.com/ | 메타 시트 |
| 구글 | https://ads.google.com/ | 구글 시트 |
| 네이버 | https://ads.naver.com/ | 네이버 시트 |
| 카카오 | (운영 결정 후) | 카카오 시트 |

> **당근 X** — KT 공식 대리점은 당근 광고 불가

### 3. 문의접수 시트 — 카톡/전화 문의 추가
- 유입채널 표준값 폰스팟과 다름 (KT 특화):
  - 메타 / 구글 / 네이버 / 카카오 / 지인 / 직접검색 / 불확실

### 4. 결제내역 — 광고비 충전

## 트러블슈팅

폰스팟 `ads/MANUAL.md` 와 동일. 단:

### KT 측에서 광고 소재 검수 거절했어
1. KT 본사 검수 의견 확인
2. 가이드라인 위반 항목 수정
3. 재집행 전 KT 검수 통과 확인

### 사전승낙서 만료
- 만료된 사전승낙서로 집행 중인 광고 즉시 OFF
- KT 본사에 갱신 요청

### "성지", "최저가" 표현 가능 여부 불명확
- KT 본사 가이드 확인 (대부분 금지)
- 대체 표현: "KT 직영", "공식 대리점", "정식 인증"

---

## 신규 광고 시작 시 체크리스트

폰스팟보다 단계 많음:

1. **KT 가이드라인 점검** — 키워드·문구·디자인
2. **사전승낙서 유효성 확인**
3. **랜딩 URL 결정** — `https://kt-directshop.com/` (KT 본사) 또는 자체 랜딩 (추후 제작)
4. **UTM 생성** — UTM_생성기 시트
   - utm_source: meta / google / naver / kakao
   - utm_campaign: 영문 소문자, 예: `kt_official_promo`, `kt_pricecheck`
5. **광고 플랫폼 UTM 등록**
6. **KT 본사 측 검수** (필요시)
7. **첫 광고 집행 + 시트 입력**
8. **2~3일 후 GA4_자동에서 source/medium 정상 인식 확인**
