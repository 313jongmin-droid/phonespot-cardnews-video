안녕하세요, 폰스팟 광교점입니다.

오늘은 5월 28일 ZDNet이 나인투파이브맥 보도를 인용해 정리한 **애플 아이폰 도난감지 자동 잠금 기능** 소식을 다룹니다.

### 핵심 요약
애플이 아이폰을 도난 당했을 때 기기가 자동으로 잠기는 신규 보안 기능을 개발 중인 것으로 전해졌습니다. 손에서 강제로 탈취됐다고 판단되면 시스템이 이를 감지해 자동으로 기기를 잠그는 방식입니다.

### 어떻게 감지하나
- **아이폰 센서**: 가속도계를 비롯한 다양한 센서 신호 활용
- **애플워치 연동**: 연결된 애플워치와 아이폰 간 거리 변화 분석
- **판단 기준**: 두 정보를 결합해 실제 도난 상황인지 확인
- **반응**: 탈취가 감지되면 아이폰이 즉시 자동 잠금 상태로 전환

안드로이드 OS의 '도난 감지 잠금' 기능과 유사한 방향입니다.

### 기존 보안 기능과의 차이
현재 애플이 제공하는 기존 보안 기능:
- **나의 찾기**: 분실·도난 시 위치 추적
- **활성화 잠금**: 초기화해도 애플 계정 없이 재사용 불가
- **도난 기기 보호**: 익숙하지 않은 장소에서 민감한 작업 시 생체 인증 요구

그러나 아이폰이 잠금 해제된 상태에서 도난 당하면 즉각적인 대응이 어렵다는 지적이 있었고, 이번 신규 기능이 이 빈틈을 메우는 방향입니다.

### 신규 기능 활성화 시 따라오는 보호
- 익숙하지 않은 장소: 저장된 비밀번호·신용카드 정보 접근 시 생체 인증 요구
- 애플 계정 비밀번호 변경 같은 민감한 작업: 최대 1시간 보안 지연 시간 적용
- 즉, 도둑이 폰을 가져가도 계정·결제 정보 탈취가 즉시 어렵게 됨

### 출시 시점
나인투파이브맥은 정확한 공개 시점은 아직 확인되지 않았지만, 입수한 코드상 현재 활발히 개발이 진행 중인 상태라고 전했습니다.

### 지금 바로 켜둘 설정
신규 기능을 기다리지 말고 지금 켜둘 보안 옵션:
- 설정 → 페이스 ID 및 암호 → '도난 기기 보호' 켜기
- 설정 → Apple ID → 나의 찾기 → '아이폰 찾기' 켜기
- 활성화 잠금은 '나의 찾기' 켜져 있으면 자동 적용
- 강력한 6자리 이상 암호 또는 영숫자 암호 설정

### 매장에서 확인할 포인트
- 도난 기기 보호 + 나의 찾기 동시 점검
- 분실·도난 시 매장 통한 대응 절차 안내
- 신규·기변별 안전한 데이터 이전 방법

정확한 공개 시점은 아직 확정되지 않았으나, 지금 바로 설정에서 도난 기기 보호와 나의 찾기를 켜두는 것이 우선입니다.
