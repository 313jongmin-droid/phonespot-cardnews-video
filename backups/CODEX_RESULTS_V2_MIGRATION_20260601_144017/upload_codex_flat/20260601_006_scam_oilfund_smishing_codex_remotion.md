안녕하세요, 폰스팟 광교점입니다.

5월 27일 행안부와 과기정통부가 **고유가지원금 2차 지급을 빙자한 스미싱 문자가 급증하고 있다**고 합동 경고했습니다. 지원금 신청 안내는 정부 공식 사이트(mygov.kr 또는 정부24)에서만 나오고, 카카오톡·문자메시지로는 절대 보내지 않습니다.

### 핵심 메시지
- **정부는 문자로 링크를 보내지 않습니다**
- 지원금 신청 안내는 mygov.kr · 정부24 등 공식 사이트 한 곳
- 5/27 행안부·과기정통부 합동 경고

### 스미싱 수법
- '고유가지원금 2차 신청 안내' 류의 문자
- 단축 URL(예: bit.ly, t.co 변형 등) 첨부
- 클릭 시 정부 사이트와 거의 똑같은 가짜 페이지로 연결
- 신청서 작성 위장한 악성 앱 설치 유도

### 악성 앱이 깔리면
- 주민번호·계좌·문자 인증번호 통째로 유출
- 정상 발신 전화도 가로채기 (사기범과 통화)
- 금융 앱 자동 로그인 정보 탈취
- 출처: 한국인터넷진흥원(KISA) 권고

### 부모님 폰 미리 차단 5가지
1. **모르는 발신자 무음**: 아이폰 → 설정 → 전화 / 갤럭시 → 전화 앱 설정
2. **후후·후스콜 설치**: 발신자 정보 자동 표시
3. **공인 백신 앱**: 알약M·V3 모바일 등 정식 앱 (가짜 백신 주의)
4. **Play 프로텍트 켜기** (안드로이드)
5. **금융 앱 생체 인증 강화**: 비번 + 지문/페이스 ID 2중

### 받았을 때 대처
- **절대 누르지 말고 바로 삭제**
- 클릭했다면 즉시 비행기 모드 켜기
- 112(경찰) + 1332(금감원) 동시 신고
- 송금했다면 1시간 안에 거래 은행 지급정지 신청
- 출처: 금감원 보이스피싱 대응 가이드

### 의심되면 직접 확인
- 카톡으로 자녀에게 캡처 전송 후 물어보기
- 정부 공식 안내는 mygov.kr·정부24·국세청 홈택스 등에서만 확인

### 매장에서 도움받기
- 부모님 폰 5가지 차단 설정 + 백신 앱 설치 + 후후·후스콜 한자리 점검
- 5월 가정의 달 마무리, 부모님께 가장 실질적인 선물
