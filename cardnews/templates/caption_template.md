## 공통 규칙

- 모든 링크는 litt.ly/phonespot (Linktree)
- 카카오톡 직링크 금지 (뽐뿌 제외)
- 광교점만 언급, 안양 해시태그 금지
- 모든 캡션 최하단에 사전승낙서 URL 필수 (한국 법령)

**사전승낙서 URL (고정)**:
```
[사전승낙서] https://ictmarket.or.kr:8443/precon/pop_CertIcon.do?PRECON_REQ_ID=PRE0000194479&YN=1
```
