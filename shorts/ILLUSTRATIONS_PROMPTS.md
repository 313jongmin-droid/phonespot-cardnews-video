# 폰스팟 일러스트 24종 ChatGPT 생성 프롬프트

## 사용 방법

1. 아래 각 일러스트별 프롬프트를 **ChatGPT(GPT-4o 또는 DALL-E)** 에 입력
2. 생성된 이미지 다운로드 (PNG)
3. **정확한 파일명**으로 `public/assets/illustrations/` 폴더에 저장
4. 24개 완료 후 `run_B_casual.bat` 빌드 시 자동 반영

## 공통 톤 가이드 (각 프롬프트에 자동 포함됨)

- 한국 휴대폰 매장 IT 뉴스 채널용 캐주얼 일러스트
- 컬러 팔레트: 오렌지(#F74B0B) 메인 + 검정(#1A1A1A) + 흰색
- 배경: 라이트 살구색(#FFF1EA) 또는 흰색
- 사이즈: 정사각형 1024×1024 또는 가로 1024×768
- 스타일: 플랫 디자인, 굵은 라인, 약간 캐주얼/B급 무드
- 한글 텍스트는 굵은 산세리프 폰트 (Pretendard 느낌)

## 통신사·브랜드 (별도 자산)

- `logos/` 폴더의 실제 로고 PNG 활용: apple, samsung, kt, lgu, skt
- 매핑: `{"type": "image", "value": "logos/<brand>.png"}`
- ChatGPT 재생성 불필요

## 동적 데이터 (React 컴포넌트, PNG 만들지 않음)

값이 슬러그마다 달라지는 일러스트는 React 컴포넌트로 처리. chunk_visuals 매핑 시 데이터 객체 전달.

| type | 데이터 형식 | 용도 |
|---|---|---|
| `calendar` | `{day, month?, label}` | 동적 날짜 (마감일/출시일 등) — 예: `{"day":"31","month":"5월","label":"마감일"}` |
| `bankaccount` | `{amount, unit?, account?, label?}` | 통장 + 잔액 — 예: `{"amount":"100,000","label":"지원금"}` |
| `pricebar` | `{label, before, after}` | 가격 비교 (출고가→실구매가) |
| `timeline` | `{items:[{date,label}, ...]}` | 다단계 일정 |
| `stat` | `{number, unit, label}` | 큰 숫자 강조 |
| `compare` | `{left:{label,value}, right:{label,value}}` | 좌우 비교 |
