# 최고의 타이포 홍보영상 (광고 의뢰를 부르는 쇼케이스) 블루프린트

> 목표: 쌓인 타이포 자산으로 "이 사람들 잘 만든다 → 우리 것도 의뢰" 를 만드는 한 편.
> 결론: **스타일 1개를 잘 고르는 게 아니라, 비트마다 최적 스타일을 배치(디렉터 컷)하고 + 오디오·폰트·그레이드로 마감**하는 것.

## 1. 핵심 원리
1. **디렉터 컷**: 단일 스타일 ✗ → 구간별 최적 스타일 배치. (엔진에 `styleMap` 추가 완료)
2. **포트폴리오 = 광고**: 메시지 전달 + 제작 실력 증명을 동시에. 보는 사람이 "저런 거 만들고 싶다"가 되게.
3. **첫 1초 훅**이 조회수·완주율을 좌우. 사운드오프(70~85%)라 자막이 메시지 운반.

## 2. 디렉터 컷 구조 (= 컴포지션 `Promo-showcase`)
`registry.ts`의 `PROMO_SHOWCASE`에 정의(자유 변경):

| 구간 | 스타일 | 이유 |
|------|--------|------|
| 오프닝 | oversize | 초대형 단어로 강한 진입 |
| 훅 | kinetic | 빠른 비트컷, 리듬 |
| 문제(말 바뀜) | glitch | 불신·변동을 시각적 텐션으로 |
| 해결(그대로 개통) | kinetic-box | 핵심 약속을 하이라이트 박스로 페이오프 |
| 신뢰(한눈에 비교) | swiss | 편집형 차분 = 믿음 |
| CTA | kinetic-box | 행동 유도 드라이브 |

→ 같은 스크립트로 조합만 바꿔 여러 버전 비교 가능(styleMap 한 줄 수정).

## 3. 엔진에서 지금 되는 것
- 구간별 스타일(`styleMap`) · `Promo-showcase` 컴포지션 등록 완료.
- 렌더: PC `npx remotion studio` → `Promo-showcase` (또는 `run_promo.bat`에 showcase 추가 가능).

## 4. "진짜 베스트"는 엔진 밖에서 완성된다 — 솔직하게
프리뷰만으론 광고 의뢰를 못 만듭니다. 아래가 체감 퀄리티의 절반 이상:
- **오디오**: 배경음악(비트) + 나레이션, 그리고 **컷을 음악 비트에 싱크**. (현재 프리뷰 무음)
- **폰트**: Pretendard 풀 렌더(프리뷰는 Noto 근사).
- **사운드 디자인**: 전환마다 whoosh·탁 효과음.
- **컬러 그레이드 통일** + 60fps 고려.
- **브랜드 스팅**: 마지막 0.5초 로고+`litt.ly/phonespot` 일관 마감.

## 5. 영상 밖 — 의뢰를 부르는 장치
- **시리즈 일관성**: 같은 톤 3~5편이 쌓여야 "잘한다"가 증명됨.
- **캡션/프로필에 "영상 제작 문의" 명시** + 연락 링크.
- 메이킹/비포애프터 살짝 노출 = "이 결을 우리가 만든다"는 신호.
