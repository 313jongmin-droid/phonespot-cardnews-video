# promo 배경음악 풀

여기에 저작권 free(상업 OK) 트랙을 넣으면 영상마다 자동 선택됩니다.

## 파일명 규칙
<무드>_<번호>.mp3   예) upbeat_01.mp3 · calm_01.mp3 · minimal_01.mp3 · confident_01.mp3

### 시작 지점 지정(선택)
파일명 끝에 `__초` 를 붙이면 그 지점부터 사용(후렴/드롭만 쓰기).
예) confident_02__70.mp3 = 1분 10초부터. 없으면 0초(처음)부터.

## 무드 (프리셋 자동 매핑)
- confident  ← showcase
- upbeat     ← punchy
- calm       ← calm
- minimal    ← data
무드별로 2~3곡 넣으면, 같은 무드 영상이라도 번호에 따라 곡이 로테이션됨(반복 방지).

## 개별 지정
스크립트(MD/JSON)에 mood 또는 music(파일명) 넣으면 그게 우선.
