### 핵심 사건
5월 25일(현지시간) 미국 전자제품 유통업체 베스트바이 직원이 고객 아이폰을 수리하던 중 사적인 사진을 보고 일부를 자신의 휴대폰으로 에어드롭한 의혹이 제기됐습니다. 해당 여성은 매장을 떠난 뒤에야 알아챘고, 베스트바이는 해당 직원이 더 이상 재직 중이 아니며 법 집행기관과 협력 중이라고 밝혔습니다.

### 폰 수리 전 점검 5가지

**1. 민감 사진 → 숨김·보안 폴더로 이동**
- 아이폰: 사진 숨김 처리 후 페이스ID·터치ID로 잠금
- 갤럭시: 보안 폴더로 옮겨 비밀번호·생체인증 접근 제한

**2. 클라우드 백업 + 초기화**
- 아이클라우드 / 삼성 클라우드에 백업
- 수리 전 기기 초기화 — 가장 확실한 방법
- 복원만으로 문제 해결 가능성 있음

**3. 주요 앱 로그아웃 + 결제 정보 제거**
- SNS·뱅킹·페이먼트 앱 로그아웃
- 등록된 카드·계좌 정보 제거
- 비밀번호 관리 — 같은 PW 재사용 X

**4. 신뢰할 매장 선택**
- 고객 앞에서 점검 가능한 매장 우선
- 애플스토어 지니어스 바 사례
- 기기가 시야에서 벗어나는 상황 최소화

**5. 수리 정책 사전 확인**
- 직원의 데이터 접근 범위
- 수리 영상 기록 여부
- 보증·환불 정책

### 광교점 폰스팟 안내
- 데이터 백업·초기화 고객 앞에서 진행
- 폰 점검·수리 시 사생활 보호 절차 안내
- 기기 교체·기변 시 데이터 안전 이전
