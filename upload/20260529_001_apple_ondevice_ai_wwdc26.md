### 핵심 요약
애플이 6월 8일 열리는 WWDC26에서 '온디바이스 AI'를 핵심 경쟁력으로 내세울 전망입니다. 아이폰·애플워치·맥에 탑재된 자체 칩이 기기 내부에서 직접 AI 쿼리를 처리하는 방식이 핵심입니다.

### 온디바이스 AI가 뭔가요
- **기존 방식**: 사용자 음성·요청 → 클라우드 서버로 전송 → AI 응답 → 다시 폰으로 받음
- **온디바이스 방식**: 사용자 요청 → 폰 안의 칩이 직접 처리 → 즉시 응답
- **차이**: 데이터가 외부 서버로 나가지 않음 + 응답 속도 빠름 + 인터넷 약한 환경에서도 동작

### 애플의 차별점
- 경쟁사들은 대규모 데이터센터 구축에 집중
- 애플은 15년간 축적한 자체 실리콘 칩 기술 기반
- 개인정보 보호 + 비용 절감을 동시에 강조

### 구글과의 협업
- 제미나이 모델을 애플 기기에서 구동할 수 있는 경량 버전으로 학습 추진
- AI 모델 경량화 기술 강화를 위해 미국 매사추세츠 소재 스타트업 '리퀴드 AI' 인수도 검토 중

### 일부는 여전히 클라우드 필요
- 대규모 모델 처리에는 구글 클라우드 + 엔비디아 기밀 컴퓨팅 활용
- 처리 과정에서 데이터·모델 암호화로 강력한 개인정보 보호
- 다만 일부 성능 저하가 발생할 수 있음

### 사용자에게 어떤 의미
- 응답 속도 향상
- 내 개인정보가 외부 서버로 나갈 일이 줄어듦
- 인터넷 약한 환경에서도 AI 기능 사용 가능
- WWDC24에서 연기됐던 AI 기능들이 이번에 다시 공개될 가능성

### 매장에서 확인할 포인트

- 아이폰 기종별 신기능 지원 여부 비교 (구형·신형 차이)
- 신규·기변별 통신3사 혜택 정리
